AiPictors DB API

GraphQL Playground

GraphQL APIでSupabaseのデータにアクセスできます。

GraphQL Playgroundを開く

利用可能なクエリとミューテーション:

クエリ例:

query {
  userLikeRankings(limit: 5) {
    id
    user_id
    count
    created_at
  }
}

ミューテーション例:

mutation {
  incrementUserLikeCount(user_id: "user123") {
    id
    user_id
    count
    updated_at
  }
}
